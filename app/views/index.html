IDM flows - READ ME

The IDM flows are hardcoded to reflect the scenario of a main ecologist using the service for the first time and creating a Defra account for the first time. The 'your application role' flow has an ultra simplified representation of the Defra account registration process. The Defra account creation screens have been included in this ultra simple format purely to test whether the user can understand that the details they enter on those screens have been pulled through into the application for the user to confirm. They are not an accurate representation of the full Defra account creation process.

The contact details section flows within the task list are hard coded to display the options a user would see if they select 'I am the main ecologist' from the 'Your application role' page.

The flows are in seperate folders but are linked together as one continuous linear journey. Work through each flow in the order presented in the task list and populate all fields inorder to make the dynamic data fields in the 'Give invoice payer details' flow work correctly.
